Passo a Passo

Cinco etapas para sair do zero a um fluxo consistente de motion graphics com IA.

1

Encontre o repositório do projeto de motion graphics

O fluxo começa com um repositório pronto que contém a estrutura base para geração de motion graphics com IA. Busque por repositórios de "Claude motion graphics" ou "AI motion board" no GitHub. O link do repositório será a fundação de todo o projeto.

Dica: Salve o link do repositório em um lugar de fácil acesso — você vai usá-lo várias vezes durante o processo.
2

Cole o link do repositório no seu assistente de código IA

Abra seu assistente de código IA preferido (Cursor, GitHub Copilot, Windsurf, etc.) e cole o link do repositório. Em seguida, peça para ele inicializar o projeto com um prompt simples como: "Initialize this project from the repo link and set up all dependencies." Aguarde os arquivos de script serem gerados automaticamente.

Dica: Se der erro na inicialização, peça ao assistente para "fix any errors and try again" — ele geralmente resolve sozinho.
3

Faça upload do vídeo ou descreva o que quer criar

Com o projeto inicializado, você tem duas opções: (1) faça upload de um vídeo existente que deseja editar com motion graphics, ou (2) descreva em texto o vídeo que quer criar do zero. Seja específico na descrição: mencione estilo, duração, tema e tipo de movimento desejado.

Dica: Quanto mais detalhada for sua descrição, mais próximo do resultado esperado você chegará na primeira tentativa.
4

Use os guias de prompt para refinar estilo e transições

O repositório inclui guias de prompt organizados por categoria: transições, estilos visuais e templates. Consulte esses guias para montar seu prompt de motion board. Combine elementos de diferentes categorias para criar um estilo único e consistente para seus projetos.

Dica: Crie seu próprio "style guide" pessoal com os prompts que funcionaram melhor — isso vai economizar muito tempo em projetos futuros.
5

Itere e refine seu processo

Os primeiros resultados raramente são perfeitos — e tudo bem. Analise o que foi gerado, identifique o que precisa melhorar (timing, estilo, transições) e ajuste seus prompts. Documente o que funcionou e o que não funcionou. Com o tempo, você vai construir um repertório de prompts que produzem resultados consistentes e de alta qualidade.

Dica: Reserve pelo menos uma semana de experimentação antes de usar em projetos de clientes. A curva de aprendizado vale muito a pena.

Prompts para Usar

Copie e adapte os prompts abaixo. Substitua os campos entre colchetes pelas suas informações.

Inicializar projeto no assistente de código

I have this repository link: [COLE O LINK DO REPOSITÓRIO AQUI]. Please initialize this project, install all dependencies, and set up the environment so I can start creating AI-generated motion graphics. Fix any errors that appear during setup.

Criar motion graphics para vídeo existente

I have a video I want to enhance with motion graphics. The video is about [DESCREVA O TEMA DO VÍDEO]. I want the motion graphics to have the following style: [ex: clean and minimal, bold and energetic, cinematic]. Please add: transitions between scenes, text animations, and visual effects that match this style. Keep it professional and polished.

Descrever vídeo do zero com motion graphics

Create a motion graphics video with the following specs:
- Theme/Topic: [SEU TEMA]
- Duration: [ex: 30 seconds, 1 minute]
- Visual style: [ex: modern, retro, corporate, creative]
- Color palette: [ex: blue and white, dark with neon accents]
- Transitions: [ex: smooth fades, dynamic cuts, slide transitions]
- Text animations: [ex: typewriter effect, fade in, bounce]
- Overall mood: [ex: professional, fun, inspiring]

Criar style guide pessoal de motion board

Based on the motion graphics we created, help me build a reusable style guide with:
1. The exact prompt parameters that produced the best results
2. The color palette and typography settings used
3. The transition styles and timing that worked best
4. A template prompt I can reuse across future projects
Format this as a structured document I can reference quickly.
