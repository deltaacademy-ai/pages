Pipeline de contas a pagar com IA · Caso Lumina

Aulas
01 AP01 — Project Contas a Pagar Lumina
Configure o Claude Project “Contas a Pagar — Lumina” com nome, descrição, System Prompt em 6 blocos e 4 arquivos de contexto (plano de fornecedores, política de conciliação, identidade visual e histórico de pedidos). Setup via interface, zero mensagens consumidas.
02 AP02 — Skill de Extração de NF
Instale a Skill nf-extractor-lumina no Project e teste com 2 NFs em PDF (uma simples, uma com armadilha didática). Descubra divergência real de R$ 47,00 do fornecedor via code execution — o princípio “LLM lê, código calcula” funcionando ao vivo.
03 AP03 — Skill de Conciliação
Instale a Skill conciliador-lumina e rode a conciliação das 2 NFs da AP02 via code execution. Aprenda a separar status de pagamento de flag de auditoria — NF pode ser conciliada e ainda carregar inconsistência registrada para investigar depois.
04 AP04 — Skill de Painel de Divergências
Instale a 3ª Skill (painel-divergencias-lumina) e gere um artefato HTML autocontido com a identidade Lumina — hero, lista de NFs e flag pendente em 3 seções de pirâmide invertida. Fecha o pipeline da A12: do PDF da NF ao painel executivo da controller.
